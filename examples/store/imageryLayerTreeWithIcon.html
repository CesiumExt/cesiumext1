This example shows how to use the CesiumExt.data.store.ImageryLayerTreeStore with the plugin CesiumExt.tree.plugin.TreeColumnIcon to show the icons for the layers:

  • This Tree Panel allows to enable/disable the Imagery Layer Visibility
  • Drag&Drop is enabled to allow to change the order of the Imagery Layers
  • It is possible to group the Imagery Layers in different groups
